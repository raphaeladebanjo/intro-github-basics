Introduction go Git Version Control System


Version Control: Version control, also known as source control, is the practice of tracking and managing changes to software code. Version control systems are software tools that help software teams manage changes to source code over time.It makes it easy to access saved codes and make adjustment when neccessary and can be consulted when the need arises.If a mistake is made, developers can turn back the clock 


Git : It is a free and open-source version control system used to handle small to very large projects efficiently. Git is used to tracking changes in the source code, enabling multiple developers to work together on non-linear development.


Process of committing a project to github repository :
 
.git init
.git add .
.git commit -m "First commit"
.git branch -m branch
.git remote add origin
